е…Ҙй—ЁзҜҮ

жҹҗеҗӣmore 2020-09-20 09:13:50 vuepress

жң¬зҜҮж–Үз« дё»иҰҒд»Ӣз»Қ vuepress е…Ҙй—ЁеҹәзЎҖдҪҝз”ЁпјҢеӯҰе®Ңжң¬зҜҮиҜҫзЁӢпјҢдҪ е°ҶдјҡеӯҰдјҡеҰӮдҪ•дҪҝз”Ёvuepress жҗӯе»әдёҖдёӘж–ҮжЎЈзұ»еһӢwebйЎөйқў

# иө·жӯҘ

еүҚжҸҗжқЎд»¶

VuePress йңҖиҰҒ Node.js >= 8.6

  1. еҲӣе»әе№¶иҝӣе…ҘдёҖдёӘж–°зӣ®еҪ•vuepress-starterпјҢwindowsдёӢзӣҙжҺҘеҸій”®еҲӣе»әеҚіеҸҜ

    mkdir vuepress-starter && cd vuepress-starter

  2. еҲқе§ӢеҢ–йЎ№зӣ®

    npm init

  3. е°Ҷ VuePress е®үиЈ…дёәжң¬ең°дҫқиө–,е®ҳж–№е·Із»ҸдёҚеҶҚжҺЁиҚҗе…ЁеұҖе®үиЈ… VuePress

    npm install -D vuepress

  4. еҲӣе»әдҪ зҡ„з¬¬дёҖзҜҮж–ҮжЎЈ

    • еңЁvuepress-starterзӣ®еҪ•дёӢеҲӣе»әdocsзӣ®еҪ•
    • еңЁdocsзӣ®еҪ•дёӢеҲӣе»әdocsзӣ®еҪ•README.mdж–Үд»¶
    • еңЁREADME.mdж–Үд»¶ еҶҷе…ҘиҮӘе·ұжғіиҰҒеҶҷзҡ„еҶ…е®№

  5. еңЁ package.json дёӯж·»еҠ еҗҜеҠЁйЎ№зӣ®й…ҚзҪ®

    иҝҷдёӘй…ҚзҪ®е№¶йқһе®ҳж–№зҡ„пјҢеҸӘжҳҜжҲ‘иҮӘе·ұд№ жғҜзҡ„vueйЎ№зӣ®

    {
  "scripts": {
    "serve": "vuepress dev docs",
    "build": "vuepress build docs"
  }
}

  6. еҗҜеҠЁжң¬ең°жңҚеҠЎ

    npm run serve

# еҹәжң¬й…ҚзҪ®

еҰӮжһңжІЎжңүд»»дҪ•й…ҚзҪ®пјҢиҝҷдёӘзҪ‘з«ҷе°ҶдјҡжҳҜйқһеёёеұҖйҷҗзҡ„пјҢз”ЁжҲ·д№ҹж— жі•еңЁдҪ зҡ„зҪ‘з«ҷдёҠиҮӘз”ұеҜјиҲӘгҖӮдёәдәҶжӣҙеҘҪең°иҮӘе®ҡд№үдҪ зҡ„зҪ‘з«ҷпјҢи®©жҲ‘д»¬йҰ–е…ҲеңЁдҪ зҡ„ж–ҮжЎЈзӣ®еҪ•дёӢеҲӣе»әдёҖдёӘ .vuepress зӣ®еҪ•пјҢжүҖжңү VuePress зӣёе…ізҡ„ж–Үд»¶йғҪе°Ҷдјҡиў«ж”ҫеңЁиҝҷйҮҢгҖӮдҪ зҡ„йЎ№зӣ®з»“жһ„еҸҜиғҪжҳҜиҝҷж ·пјҡ

.
в”ңв”Җ docs
в”Ӯ  в”ңв”Җ README.md
в”Ӯ  в””в”Җ .vuepress
в”Ӯ     в””в”Җ config.js
в””в”Җ package.json

дёҖдёӘ VuePress зҪ‘з«ҷеҝ…иҰҒзҡ„й…ҚзҪ®ж–Үд»¶жҳҜ .vuepress/config.jsпјҢе®ғеә”иҜҘеҜјеҮәдёҖдёӘ JavaScript еҜ№иұЎпјҡ

module.exports = {
  title: 'Hello VuePress', # зҪ‘з«ҷж Үйўҳ
  description: 'Just playing around' # зҪ‘з«ҷжҸҸиҝ°
}

еҜ№дәҺдёҠиҝ°зҡ„й…ҚзҪ®пјҢеҰӮжһңдҪ иҝҗиЎҢиө· npm run serveпјҢдҪ еә”иҜҘиғҪзңӢеҲ°дёҖдёӘйЎөйқўпјҢе®ғеҢ…еҗ«дёҖдёӘйЎөеӨҙпјҢйҮҢйқўеҢ…еҗ«дёҖдёӘж Үйўҳе’ҢдёҖдёӘжҗңзҙўжЎҶгҖӮVuePress еҶ…зҪ®дәҶеҹәдәҺ headers зҡ„жҗңзҙў вҖ”вҖ” е®ғдјҡиҮӘеҠЁдёәжүҖжңүйЎөйқўзҡ„ж ҮйўҳгҖҒh2 е’Ң h3 жһ„е»әиө·дёҖдёӘз®ҖеҚ•зҡ„жҗңзҙўзҙўеј•гҖӮ
